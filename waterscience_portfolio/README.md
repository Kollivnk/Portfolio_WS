# WaterScience | Development Portfolio

A premium, high-impact portfolio website showcasing 12 specialized Odoo modules developed for WaterScience.

## Features
- **Theme Toggle**: Advanced Dark/Light mode switcher with persistent user preference.
- **Classy & Modern UI**: Premium glassmorphism design with animated background depth.
- **Professional Typography**: Uses Plus Jakarta Sans for body and Playfair Display for elegant headings.
- **Detailed Module Catalog**: 12 modules explained point-wise across Sales, CRM, HR, and Operations.
- **Dynamic Interaction**: Scroll-triggered reveals, staggered card animations, and a refined glass-visual hero.
- **Responsive & Optimized**: Seamless experience from high-end desktops to mobile devices.

## How to Deploy to Vercel

1. **GitHub/GitLab/Bitbucket**: Push this folder to a repository.
2. **Vercel Project**: Import the repository into Vercel.
3. **Deploy**: Vercel will automatically detect the static files and provide a live URL.

Alternatively, you can drag and drop the `waterscience_portfolio` folder directly into the Vercel dashboard for instant deployment.

## Presentation Strategy
- **Overview**: Start with the hero section to highlight the "Innovation in WaterScience" theme.
- **Functional Deep Dive**: Scroll through the three main categories. Each card now has bullet points to help you explain technical details during the presentation.
- **Style Highlight**: Point out the "glassmorphism" cards and animated background as a reflection of modern development standards.
